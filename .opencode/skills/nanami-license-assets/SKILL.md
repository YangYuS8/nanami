# Nanami License and Assets Skill

## 原则

1. 不提交未确认许可的 Live2D 模型。
2. 不提交 Live2D Cubism Core 受限二进制，除非许可证允许。
3. 引入 AmaiGirl 代码时必须保留 Apache-2.0 notice，并标记修改。
4. 第三方素材必须记录来源、许可证、是否可商用、是否可再分发。
5. 示例模型和默认模型应分离，避免仓库无法合法发布。
6. 不要提交字体文件，除非许可证明确允许再分发。
