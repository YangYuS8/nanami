# Nanami Permission Review Skill

任何危险能力必须经过权限审查。

## 权限等级

- L0：只聊天
- L1：读取剪贴板、截图、当前窗口信息
- L2：读取指定项目目录
- L3：写入指定项目目录
- L4：执行本地命令
- L5：挂载宿主机目录到 CubeSandbox
- L6：联网、下载依赖、访问外部服务
- L7：删除文件、覆盖配置、修改系统状态

## 审查问题

1. 这个操作是否可能泄露隐私？
2. 是否可能修改用户文件？
3. 是否可能执行任意命令？
4. 是否可能联网？
5. 是否可能影响系统配置？
6. 是否能限制到本次任务？
7. 是否需要审计日志？
8. 是否需要撤销/回滚？

## 默认策略

- 默认拒绝危险操作。
- 优先在 CubeSandbox 执行不可信代码。
- 写入宿主机前必须展示 diff。
- 删除文件必须二次确认。
